# Curated External Skills

Skills I use frequently that are not authored by me.

## Frontend Design

- Source: https://github.com/anthropics/skills
- Install:
  - `npx skills add https://github.com/anthropics/skills --skill frontend-design`
- Why I use it:
  - High-quality UI/UX output for production-style frontend interfaces.
- Best for:
  - Landing pages, polished app UIs, visual refactors.

## Webapp Testing

- Source: https://github.com/anthropics/skills
- Install:
  - `npx skills add https://github.com/anthropics/skills --skill webapp-testing`
- Why I use it:
  - Practical browser interaction and debugging workflow for local web apps.
- Best for:
  - UI verification, reproducing frontend bugs, screenshots/log-based debugging.

## Find Skills

- Source: https://github.com/vercel-labs/skills
- Install:
  - `npx skills add https://github.com/vercel-labs/skills --skill find-skills`
- Why I use it:
  - Fast discovery of available skills when starting a new project.
- Best for:
  - Skill scouting, setup bootstrap, quickly finding fit-for-purpose skills.
